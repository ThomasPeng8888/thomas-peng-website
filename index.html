個人簡介和申請動機

大家好！我是彭宏紳，28歲，畢業於逢甲大學統計研究所，現任「財團法人雅文兒童聽語文教基金會」的數據分析師。我的主要工作內容包括協助會內營運數據規格化、從 SQL 資料庫中抽取營運數據並製作成 BI 報表，以及與各部門合作建構常態業務觀測指標，從數據中提出洞察與建議。

在我的工作中，我經常需要處理捐款原始明細，並將其轉換為適合批次上傳到系統的格式。為了提升效率，我學習並應用 Flask 框架，開發了一個自動化的數據轉換工具。這項經歷不僅提升了我的工作效能，也激發了我對程式開發的濃厚興趣。從中，我發現自己樂於運用技術解決實際問題的成就感。

其實，我在求學階段已接觸過程式語言，對於 Python 和 R 有初步的認識，但尚未深入專精。工作中與 AI 工具的互動，讓我逐漸體會到程式開發的無限可能。我期望有一天能夠完全靠自己構建大型架構，AI 僅作為輔助工具，幫助我更快完成 debug。

因此，我希望能參與這次的訓練計劃，不僅提升技術能力，也為未來的職涯發展創造更多可能性。我相信這次學習將幫助我突破現有的技術水平，開啟一條更專業、更自信的軟體開發道路。

學習經歷

在求學階段，我主修統計學，對於數據分析與模型建構有紮實的基礎，這些知識不僅讓我具備處理大數據的能力，也讓我對於數據背後的故事產生濃厚的興趣。在學術研究中，我曾參與多個專案，負責清理數據、進行分析並撰寫研究報告，這讓我對數據的價值有了更深的體認。

此外，我在大學時期初次接觸程式設計，學習了 Python 和 R 語言，完成過幾個小型專案，例如製作簡單的資料視覺化工具、撰寫自動化數據處理腳本等。這些經歷讓我對程式邏輯有了初步的了解，但也發現自己需要進一步強化技術能力。

研究所期間，我進一步深化了統計與機器學習的相關知識，同時學習如何運用 SQL 整合和管理大型資料集。當時的學術研究經歷，讓我培養出細心與耐心的工作態度，並理解專案管理與多方協作的重要性。

進入職場後，為了解決實際業務需求，我自學 Flask 框架，結合工作中遇到的挑戰開發小型工具。例如：透過建立資料格式轉換的自動化程式，縮短了流程時間並減少人工錯誤率。這段自學經歷讓我明白學習是一個持續的過程，透過實踐，我逐漸發展出對程式開發的興趣與信心。

未來，我希望能進一步專攻全端開發技術，結合數據分析的專長，打造能真正解決受眾問題的創新解決方案。

學習時間安排

由於目前我在「雅文兒童聽語文教基金會」擔任數據分析師，工作內容繁忙且具有挑戰性，因此每天的時間安排需要更為精確和高效。儘管如此，我深知持續學習與自我成長對於進一步提升我的專業能力至關重要。因此，在繁重的工作之餘，我會非常重視學習時間的安排。

每天返家後，我會安排固定的 2 小時學習時間，這段時間內，我會專注於深入理解當天所學的技術和理論，並進行實際操作。例如，我會利用這段時間練習程式編寫、參加線上課程、或者讀取技術書籍與文章。這樣的學習安排不僅讓我能夠緊跟技術發展的步伐，也能夠幫助我解決日常工作中遇到的技術問題。

而在週末的兩天，我會額外安排約 8 小時的學習時間，進行更系統化的複習與專案實作。週末的學習不僅讓我有時間鞏固平日所學的內容，還能夠專注於一些較為複雜的專案或實作練習，這些練習有助於我將學到的知識應用於實際問題中，提升解決問題的能力。

除了固定的學習時間，我也會定期進行學習計畫的回顧與調整，確保學習進度與效果。我深信只有通過持續的學習與實踐，才能真正提升自己，並將所學的技術應用於工作中，為職業發展鋪平道路。

這樣的學習安排雖然要求我平衡好工作、學習與生活，但我對技術的熱情和對未來的期望，讓我能夠保持高度的動力，並不斷挑戰自己，迎接每一個學習的機會。希望能夠透過這樣的努力，為自己未來的職業生涯奠定更加堅實的基礎。

投入領域的選擇與回報

軟體技術日新月異，選擇適合自己的領域確實是一個需要深思熟慮的過程。對於前後端技術的學習，我的選擇基於以下幾點考量：

首先，前後端開發是軟體開發領域中非常重要且基礎的技術。前端技術（如 HTML、CSS、JavaScript）讓我們能夠直接與用戶交互，設計友好的介面以提升用戶體驗；而後端技術（如 Node.js、Python Flask、Django）負責處理伺服器邏輯、資料庫和系統安全等後台操作。兩者結合能讓一名開發者具備全面的能力，從而應對更多元的工作需求。

其次，學習這些技術能大大提高我的職業競爭力。從就業市場的角度看，無論是初創公司還是大企業，都對熟悉全端開發的工程師有著很高的需求。而對於像我這樣的初學者而言，掌握前後端技術，可以讓我更快上手專案開發，並能適應不同團隊和技術堆棧的需要。

再者，我認為選擇這個領域還是基於其可觀的回報——無論是薪資回報還是個人成就感。全端開發的工作不僅薪酬待遇可觀，還能讓我參與更有創造力的專案，比如從頭到尾構建一個完整的應用程式，看到自己的設計和邏輯落地，這無疑是極大的成就感來源。

最後，前後端技術有豐富的資源與社群支持。無論是通過網上的免費教學資源，還是參與技術社群中的討論，都能讓我在學習中少走彎路，甚至有機會接觸到更多的實戰項目。

因此，無論從市場需求還是技術應用的廣度來看，前後端技術的學習都是一個值得投入且回報豐厚的選擇。希望透過不斷的學習與實踐，我能在這條道路上穩步前行。

處理負面情緒的經驗

面對情緒管理較不佳的主管，有時候會感到生氣或挫折，特別是當某些言語對我造成心理內耗時，這種情緒確實難以忽視。但我通常會採取積極的方式來處理。首先，我會選擇與朋友或家人傾訴，將情緒釋放出來，這讓我感到被理解與支持。

其次，我嘗試將這些負面情緒轉化為動力，提醒自己不斷進步並成為更好的人。為了避免被情緒左右，我還會學習正念練習或運動來放鬆心情，幫助自己更專注於手頭的工作。同時，我也反思這些經歷，試著從中學習到溝通技巧或情緒控制的方法，讓自己在面對未來類似情況時更具韌性。

此外，我逐漸意識到，每個人都有各自的壓力來源，而這些壓力可能反映在他們的表達方式中。因此，我學會用更多的同理心看待這些互動，嘗試理解對方的處境，從而減少自己對言語的過度解讀。這不僅讓我能更理性地面對挑戰，也幫助我在職場中建立更穩健的心理素質。

最重要的是，我相信每一次的困境都是一個讓自己成長的契機，只要保持學習與反思的態度，我就能從中找到方法讓自己變得更強大。

開發技術心得

作為一個剛開始學習開發的初學者，這段時間的學習過程讓我深刻體會到程式開發的挑戰與成就感。在這段時間內，我學習了如何透過 GitHub Pages 快速部署靜態網站，並且體驗到了編寫 HTML、CSS 以及基本 JavaScript 的樂趣。

剛開始學習時，我對於前端開發的理解還停留在很基礎的層次。HTML 的標籤結構、CSS 的樣式設定，這些對我來說都是全新的領域。我花了不少時間來理解這些語法的運作方式，尤其是在設計頁面時，如何讓網站的外觀既美觀又實用，這是一個不斷摸索的過程。

當我第一次成功地將自己的靜態網站部署到 GitHub Pages 上時，內心充滿了成就感。這讓我意識到，雖然從一開始對這些技術感到陌生，但只要持續學習並實踐，就能逐漸掌握。

這次的學習過程不僅提升了我對前端技術的理解，也讓我開始對軟體開發充滿了興趣。我發現自己在學習中越來越有信心，並且在解決問題的過程中享受到了許多樂趣。雖然這只是一個小小的開始，但它讓我更堅定了繼續學習、挑戰更高層次技術的決心。

未來，我希望能夠繼續深化對全端開發的學習，並將這些技術應用到更多實際的專案中。我相信，只要持續努力，我可以在這條技術的道路上越走越遠。

工作與社會的連結

作為非營利組織的數據分析師，我認為我的工作與整個社會群體之間有著密切的連結關係。我的主要職責是分析和整理來自各種來源的數據，提供有價值的洞察和決策支持。這些數據並不僅僅是冷冰冰的數字，而是代表著個體、社區及整個社會系統的狀況和需求。

在非營利領域，我所處的組織通常以改善社會問題、提升社會福祉為使命。我的工作幫助組織瞭解受益者的需求、評估項目的影響力，並將這些結果轉化為可操作的建議。這種洞察不僅能夠幫助組織更有效地運營，也能讓我們更好地理解社會問題的核心，並找到更具影響力的解決方案。

社會群體的需求是多樣且不斷變化的，而我的數據分析工作能夠橋接這些需求與資源之間的差距。例如，通過對捐贈者行為的分析，我能夠提供哪些活動或政策能夠吸引更多的支持者，進而提高資源的分配效率。這不僅是支持組織運營的需要，也是對整個社會群體的貢獻，因為我們的工作最終能夠幫助更多需要幫助的人。

此外，數據本身是強大的工具，可以幫助我們更客觀、更精確地看待社會問題。這讓我深刻意識到，自己不僅是數據的解讀者，也是在社會改變中發揮作用的一員。每一份報告、每一次的數據分析，都能為組織的戰略決策提供依據，並且影響到那些受益者的生活。從這個角度來看，我的工作直接與整個社會群體的福祉息息相關。
